# Processing of SMART data

This folder contains scripts which processes the raw data acquired by the five SMART spectrometers, including dark current and cosine correction. 

`00_smart_vis_iup.pro` --> SMART VIS spectrometer upward radiance
`00_smart_vis_fdown.pro` --> SMART VIS spectrometer downward irradiance
`00_smart_vis_fup.pro` --> SMART VIS spectrometer upward irradiance
`00_smart_nir_fdown.pro` --> SMART NIR spectrometer downward irradiance
`00_smart_nir_fup.pro` --> SMART NIR spectrometer upward irradiance
`einlesen_vis_polar.pro` --> reader raw files (SMART VIS spectrometer)
`einlesen_pgs_polar.pro` --> reader raw files (SMART NIR spectrometer)
`config_quicklook.txt` --> Path of calibration and data files
